# 두 개의 스켈레톤 스트림으로 3D 스켈레톤 재구성

Parent: #<mpv-v2-umbrella-issue-number>

## 요약
두 개의 영상에서 각각 추출한 스켈레톤 스트림을 입력으로 받아, 하나의 3D 스켈레톤을 재구성하는 최소 파이프라인을 만든다.

## 목표
- 두 입력 스켈레톤 스트림을 같은 시간축으로 정렬한다.
- 두 시점 정보를 이용해 하나의 3D 스켈레톤 포즈를 추정한다.
- 이후 calibration, smoothing, robustness 작업의 기반을 마련한다.

## 범위
- 두 스켈레톤 입력 포맷을 정의하거나 정규화한다.
- 프레임 동기화 규칙을 정한다.
- 관절 인덱스 및 좌우 매칭 규칙을 정한다.
- 최소 수준의 3D 관절 재구성 경로를 구현한다.
- 결과를 확인하거나 시각화할 수 있는 기본 출력을 만든다.

## 제외 범위
- 고급 smoothing 또는 filtering
- 최소 구현 범위를 넘는 occlusion 처리
- 다인원 지원
- 실서비스 수준 정확도 최적화
- UI polish

## 성공 조건
- 두 개의 스켈레톤 입력을 받아 하나의 3D 스켈레톤 출력을 생성할 수 있다.
- 핵심 관절 좌표가 일관된 구조로 출력된다.
- 최소 1개 이상의 샘플 입력 쌍으로 결과를 재현할 수 있다.
- 현재 한계와 다음 리스크가 문서화된다.

## 비고
- 1차 목표는 최고 정확도가 아니라 파이프라인 성립 여부 검증이다.
- 후속 이슈에서 calibration, smoothing, robustness를 분리해 개선한다.
