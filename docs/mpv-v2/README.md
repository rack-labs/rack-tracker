# MPV v2 Docs

This directory is the active planning workspace for `mpv v2`.

## Structure

- `issues/README.md`: how to manage v2 issue documents
- `issues/umbrella/`: umbrella issue drafts and shared tracking notes
- `issues/sub-issues/`: scoped implementation and chore issue drafts

## Working Rule

- Put new v2 scope, sequencing, and cleanup decisions here.
- Treat `docs/mvp-v1/` as reference material, not as the place for new v2 planning.
