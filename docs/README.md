# Docs Index

`docs` 디렉토리는 목적과 작업 단위 기준으로 아래처럼 나눈다.

- `features/`: 기능별 명세, 아키텍처, 이슈 초안, 의사결정 문서
- `process/`: 협업 규칙, 커밋/푸시 워크플로우
- `planning/`: TODO, 후속 작업 관리
- `product/`: UI, 제품 기획 문서
- `data/`: 데이터 구조 해설, 샘플 데이터 문서
